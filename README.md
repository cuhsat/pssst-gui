Pssst HTML
==========
A simple `HTML` interface for [Pssst](https://github.com/cuhsat/pssst).

Usage
-----
```
$ pssst-html [option|port]
```

Options
-------
* `-h` Shows the usage
* `-l` Shows the license
* `-v` Shows thw version
